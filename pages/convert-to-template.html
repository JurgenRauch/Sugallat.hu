Header Template Converter

This tool helps you convert existing pages to use the new header template system.

Step 1: Remove Navigation from Pages

Remove the entire navigation section from each HTML file:

<!-- Navigation --> <nav class="navbar"> <div class="nav-container"> ... (entire nav section) </div> </nav>

Step 2: Add Header Loader Script

Make sure each page includes the main.js script (which now includes header loading):

<script src="js/main.js"></script>

Step 3: Update Language Switcher

Update the language switcher links in each page to point to the correct language versions:

<div class="language-switcher"> <a href="current-page.html" class="lang-link active">HU</a> <span class="lang-separator">|</span> <a href="en/english-page.html" class="lang-link">EN</a> </div>

⚠️ Important Notes:

  • The header.html file must be in the root directory
  • All pages must be served from a web server (not file://) for fetch() to work
  • Test the navigation on each page after conversion
  • Make sure the active states work correctly

Benefits of This Approach:

  • ✅ Single source of truth for navigation
  • ✅ Easy to update navigation across all pages
  • ✅ Consistent navigation structure
  • ✅ Automatic active state management
  • ✅ Reduced code duplication
